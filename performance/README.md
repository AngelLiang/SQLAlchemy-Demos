# Performance

各种 SQLAlchemy 用例的性能分析套件。

- 批量插入（bulk_inserts.py）
- 具有或不具有事务的单个插入
- 获取大量的rows（large_resultsets.py）
- 进行大量简短的查询（short_selects.py）

## 快速开始

```
python -m performance --help
```

```
python -m performance bulk_inserts
```
